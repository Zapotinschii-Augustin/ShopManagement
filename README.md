# Aplicatie de gestionare a unui magazin
### Functii ale aplicatiei ⛺
Pentru Management:
- Adăugare produsului;
- Editare/ștergere produului;
- Afișare lista produse;
- Căutare produse;
- Sortare produse;

Pentru Clienti:
- Cumparare produse

### Biblioteci suplimentare folosite
- System.Reflection
- System.Collections.Generic
- System.IO
